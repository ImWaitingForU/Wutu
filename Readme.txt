已确定要用到的依赖：(版本可变)

图片加载：picasso   compile 'com.squareup.picasso:picasso:2.5.2'
网络请求：volley     compile 'com.android.volley:volley:1.0.0'

关于代码规范：

    1见名知意
    2方法使用文档注释(/** xxx **/),必要时标明参数返回值等
    3全局变量使用mXXX,静态使用sXXX,常量大写，单词之间用下划线分割_
    4其他遵循基本注释原则
